## 安装GIT

因为我用的是Xcode做的开发，git是Xcode自带的。如果没有安装git的请自行百度安装方法。

## 创建仓库

git 的仓库分为 本地仓库 和 远程仓库。 仓库的英文名为repository。 在仓库中我们可以进行文件的修改，删除，更新 。git 都能能够记录下来。 git 的好处还有，我们可以会滚到某一次提交的版本。或者是某一分支。
